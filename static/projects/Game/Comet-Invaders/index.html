In this little Prototype for an arcade shooter you must defend Earth from incoming comets. It was my final project for the Harvard CS50 course. For learning purposes, it is written in plain C language, which required me to handle memory management and collision detection myself. I used the Raylib library only to implement the graphics and sounds. I created the graphics in Blender, Substance Painter and Affinity Photo.

You can play it in your web browser or download it at: polyerror.itch.io/comet-invaders
