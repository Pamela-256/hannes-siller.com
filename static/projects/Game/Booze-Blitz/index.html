This was our entry for the Spelkollektivet Midsummer Game Jam in Sweden, where it won 3rd prize for "Best Game Mechanic." Antonio Ackalin implemented the (drunken) game mechanic using physics (and other parts), Michael Campos implementing the main menu (and other parts). The 3D assets are created by Axel Magnuson and me using Blender. We used the Unity Engine and game design and Concept where a collaborative effort of the whole team.
It was a one-week, in-person game jam with over 60 participants. I enjoyed the environment so much that I decided to extend my stay to three months. Spelkollektivet is a unique space where you can book a room and shared office space alongside a rotating community of 50 to 70 game developers from all over the world. Some stay for a few days, while others stay for years. Some, like me, are just at the beginning of their journey, while others have already shipped multiple games on Steam.
It is an incredible hub for indie game development. There is a small game conference held every week where residents present their current progress and learnings. These sessions are usually streamed every Thursday on the Spelkollektivet Twitch channel.
